删除GitHub上上传的文件不是整个项目方法

删除GitHub上上传的文件不是整个项目方法：
1.在你的本地目录下手动删除你要删除的东西
2.在客户端输入：git commit -a -m “A file was deleted”
3.在客户端：git push
即可删除github上上传的文件！
